# Hello World Lorem Ipsum

This repository contains a tiny Bash script that prints "Hello, World!" followed by a short Lorem Ipsum paragraph in the terminal.

Files:
- `hello_lorem.sh`: Bash script to run.

Run (any of these):

```bash
# run with bash explicitly
bash hello_lorem.sh

# or make it executable then run
chmod +x hello_lorem.sh
./hello_lorem.sh
```

Requirements: `bash` (available on Ubuntu by default).
